# BreatheFree

BreatheFree is a responsive smoking-cessation support experience built with React and Vite. The interface is presented under the **Clearway** product identity and helps users track cravings, celebrate recovery milestones, and use short breathing resets.

## Features

- Three-step onboarding and quit-plan setup
- Daily smoke-free streak and recovery dashboard
- Craving intensity and trigger tracking
- Guided two-minute breathing reset
- Progress metrics, achievements, and accountability chat
- Responsive desktop and mobile layouts

## Run locally

```bash
npm install
npm run dev
```

Open the local address shown by Vite.

## Production build

```bash
npm run build
```
